# Activity 6.1: Providing Input Validity Feedback upon Form Submission

Nesta atividade, você criará um formulário básico que permite aos usuários inserir um endereço de e-mail e uma senha. A entrada fornecida em cada campo de entrada é validada e o resultado da validação é armazenado (para cada campo de entrada individual).

O objetivo é adicionar um estilo geral ao formulário e um estilo condicional que se torna ativo quando um formulário inválido é enviado. Os estilos exatos ficam a seu critério, mas para destacar os campos de entrada inválidos, a cor de fundo do campo afetado deve ser alterada, assim como a cor da borda e a cor do texto do rótulo correspondente.

Os passos são os seguintes:

- Crie um novo projeto React e adicione um componente de formulário a ele.
- Insira o componente de formulário no componente raiz do projeto.
- No componente de formulário, insira um formulário com dois campos de entrada: um para inserir um endereço de e-mail e outro para inserir uma senha.
- Adicione rótulos aos campos de entrada.
- Armazene os valores inseridos e verifique sua validade após o envio do formulário (você pode usar a criatividade para criar sua própria lógica de validação).
- Selecione as classes CSS apropriadas do arquivo index.css fornecido (alternativamente, você também pode escrever suas próprias classes).
- Adicione-as (as classes) aos campos de entrada inválidos e seus respectivos rótulos assim que valores inválidos forem enviados.
